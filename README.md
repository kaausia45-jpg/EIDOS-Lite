# EIDOS Lite GUI (v9.2)
Emotion-Control Interface for the EIDOS Framework

## Overview
This repository provides a PyQt-based GUI interface that visualizes and manipulates
emotional state vectors for the EIDOS framework.  
The core cognitive modules (reasoning, self-reflection, and abduction systems) are **not included**.

## Features
- Real-time emotion visualization (12-dim vector)
- Manual emotion override and sync
- Goal dashboard and proactive speech placeholders
- Character widget animation (PyQt + Pygame)

## Disclaimer
EIDOS Lite does **not** contain or expose the proprietary cognitive core.
All cognitive and reasoning logic has been replaced with stubs for demonstration.
